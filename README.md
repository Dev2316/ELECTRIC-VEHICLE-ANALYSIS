# Electric Vehicle Analysis

This project analyzes electric vehicle population data to identify important trends, patterns, and insights related to EV adoption. The analysis was carried out as a group academic project and includes a detailed report, presentation, and Tableau dashboard/workbook for visualization.

## Project Overview

The purpose of this project is to explore electric vehicle population data and understand how EV adoption is changing across different categories such as vehicle type, make, model, location, and eligibility factors.

This project helps answer questions such as:
- Which EV brands and models are most common?
- How is EV adoption distributed across regions?
- What trends can be observed in electric vehicle growth?
- How can visualization tools help communicate EV-related insights?

## Objectives

- Analyze electric vehicle population data
- Identify major trends and patterns in EV adoption
- Create meaningful visualizations for better understanding
- Build an interactive Tableau dashboard
- Summarize findings in report and presentation format
